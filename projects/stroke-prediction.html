Stroke Prediction Using Machine Learning

Role: Sole Developer

Project Description

This project focuses on predicting stroke risk using a healthcare dataset through supervised machine learning techniques. The goal was to build an accurate and interpretable classification model while addressing real-world data challenges such as noise and feature redundancy.

The dataset was preprocessed and analyzed to identify the most relevant features. A K-Nearest Neighbors (kNN) classifier was implemented and optimized, with additional improvements achieved using dimensionality reduction techniques.

Methodology

  • Data cleaning and preprocessing of healthcare records
  • Feature selection and engineering
  • Stroke classification using K-Nearest Neighbors (kNN)
  • Dimensionality reduction using PCA and LDA
  • Performance evaluation across multiple feature sets

Results

The final model achieved robust classification performance, demonstrating improved accuracy and class separation through the use of PCA and LDA, and careful tuning of training and testing strategies.

Skills & Deliverables

  • Feature Engineering
  • Data Preprocessing
  • NumPy
  • pandas
