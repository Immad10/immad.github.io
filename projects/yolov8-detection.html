Real-Time Object Detection with YOLOv8

Role: Sole Developer

Project Description

This project implements a real-time object detection pipeline using the YOLOv8 deep learning model. The system processes video input frame by frame, detects objects with high confidence, and draws labeled bounding boxes around each detected object in real time.

The lightweight yolov8n.pt model is used to ensure fast inference while maintaining strong detection accuracy. The annotated output is saved as a new video file, making the system suitable for offline analysis as well as near real-time applications.

This solution is ideal for use cases such as surveillance systems, traffic monitoring, and visual analytics of recorded footage.

Tech Stack

  • Model: YOLOv8 (Ultralytics)
  • Programming: Python
  • Libraries: OpenCV, Ultralytics YOLO
