Astronomical Time Series Classification for PLAsTiCC

Role: Sole Developer

Project Description

This project presents a machine learning solution to the Photometric LSST Astronomical Time series Classification Challenge (PLAsTiCC). The objective is to classify astronomical objects into 15 distinct classes based on their photometric light curves.

Since the input data consists of variable length time series, the pipeline applies extensive feature engineering to transform raw light curves into fixed length feature vectors. Multiple robust machine learning models are trained and evaluated to achieve strong generalization performance.

Methodology

  • Preprocessing of multi-band photometric time series data
  • Feature engineering to convert light curves into fixed length vectors
  • Training using HistGradientBoostingClassifier
  • Ensemble modeling with Random Forest, LightGBM, and Logistic Regression
  • Model evaluation using probabilistic classification metrics
