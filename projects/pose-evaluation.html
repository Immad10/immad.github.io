Pose Detection & Evaluation System

Role: CV Expert | Programming

Project Description

This project implements a Pose Detection and Evaluation System using MediaPipe and OpenCV. The system detects human body poses from video files or live webcam feeds and evaluates pose accuracy by comparing detected landmarks with predefined ground truth coordinates.

Pose accuracy is computed using Euclidean distance error metrics. Results are visualized in real time and saved for offline analysis, making the system suitable for both research and applied computer vision tasks.

Key Features

  • Human pose detection from video and live webcam streams
  • Landmark comparison with ground truth coordinates
  • Pose accuracy evaluation using Euclidean distance
  • Real-time visualization and annotated video generation
  • Export of pose errors to CSV files and sample frame images

Applications

  • Sports performance analysis
  • Physical therapy and rehabilitation tracking
  • Motion evaluation for animation and robotics
