White Blood Cell Image Segmentation and Analysis

Role: Sole Developer

Project Description

This project focuses on segmenting and analyzing white blood cells from microscopic blood smear images using classical image processing and unsupervised learning techniques. The pipeline is designed to accurately isolate cellular components for quantitative medical image analysis.

Multiple preprocessing and segmentation strategies are applied to handle staining variations, noise, and overlapping structures commonly present in microscopic imagery.

Methodology

  • Precise segmentation of nucleus and cytoplasm from stained blood smear images
  • Noise reduction and refinement using morphological operations
  • HSV color space analysis and intensity histogram based feature enhancement
  • K-Means clustering for robust separation of cellular regions
  • Quantitative evaluation using Dice Coefficient against ground truth masks

Skills & Deliverables

  • Python
  • OpenCV
  • NumPy
  • Matplotlib
