Track Football Players in a Video and Analyze Their Performance

Project Description

Developed a computer vision system to detect, track, and analyze football players from match footage using YOLOv8 and ByteTrack. The system tracks each player across frames and computes performance metrics such as total distance covered, average speed, and activity ratio.

The pipeline generates annotated videos and analytical outputs that can be used for performance evaluation, coaching insights, and sports analytics applications.

Key Features

  • Real time player detection and tracking
  • Distance, speed, and activity analysis
  • Heatmap and movement pattern extraction
  • Annotated video generation

Tools & Technologies

  • Programming: Python
  • Computer Vision: OpenCV
  • Models: YOLOv8, ByteTrack
  • Libraries: NumPy, Ultralytics
