Predictive Modeling & Classification Using Machine Learning

Role: Sole Developer

Project Description

This project demonstrates a machine learning pipeline applied to a global health dataset for predictive analytics and classification. The objective was to model life expectancy trends and classify countries based on development status.

The workflow includes data cleaning, feature engineering, model selection, training, evaluation, and result interpretation, highlighting practical regression and classification techniques.

Key Tasks

  • Cleaned and engineered features from a global health dataset
  • Built regression models (linear and polynomial) for life expectancy prediction
  • Performed forecasting on future data points
  • Classified countries as Developed or Developing
  • Evaluated models using standard performance metrics

Tech & Tools Used

  • Python (Pandas, NumPy, Matplotlib, Seaborn, Scikit learn)
  • Regression and classification techniques
  • PolynomialFeatures, StandardScaler, Pipelines
  • Data visualization and predictive analytics

Skills & Deliverables

  • Machine Learning
  • Python
  • Data Preprocessing
  • pandas
  • NumPy
