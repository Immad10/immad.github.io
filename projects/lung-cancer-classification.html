Lung Cancer Classification Using Xception Transfer Learning

Role: Sole Developer

Project Description

This project presents an AI based system for automated lung cancer classification from CT scan images using Xception transfer learning. The goal is to support early diagnosis by accurately distinguishing between multiple lung cancer types and normal cases.

The model processes high resolution CT images and applies data augmentation, learning rate scheduling, and early stopping to improve generalization and prevent overfitting. Model interpretability is enhanced using Grad-CAM visualizations.

Key Features

  • Classification of four classes: Normal, Adenocarcinoma, Large Cell, Squamous Cell
  • Transfer learning using the Xception architecture
  • Input image resolution of 350√ó350 pixels
  • TensorFlow based data augmentation
  • Learning rate scheduling and early stopping
  • Grad-CAM heatmap visualization for explainability

Results

The proposed model achieves a high validation accuracy of 96.2%, demonstrating strong performance and robustness across multiple lung cancer categories.
